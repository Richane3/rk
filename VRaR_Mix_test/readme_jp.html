説明：

セキュリティのため、ブラウザーがFlashかWebGLの含んだローカルのコンテンツを再生するのを制限しています。 閲覧するには出力されたデータ全体をサーバーにアップロードして、URLより再生する必要があります。 サーバーの所有していない場合、手動で出力されたデータを弊社のプラットフォームEP-Skyにアップロードすることができます。若しくはVRTourMakerの「出力設定」画面よりEP-Skyに同期することができます。。

VRツアーを閲覧するには

本VRツアーはWebGL技術を基づき制作されたものです。WinとMacのコンピュータより再生するにはIE11.0、Safari5.0、Chrome21、Firefox50以上のバージョンを利用する必要があります。iPhoneはiPhone6以上にしてください。AndroidはAndoroid OS 5.0以上にしてください。

どのようにVRツアーを自分のウェブページに埋め込みますか？

  1. 下記のスクリプトをご利用のウェブページにコピー＆ペーストしてください。 
    <a href="#" onClick="javascript:window.open(
'pano.html',
'_blank',
'status=yes,top=0,left=0,width=640,height=480');">
パノラマネーム
</a>
    ご注意：「pano.html」 はご利用のページの名前にリネームする必要があります。
  2. VRTourMakerより出力されたデータを御社のサーバーにアップロードして、URLより再生して結構です。
    ご注意：フォルダーにあるファイルの構成は変更しないでください。アップロードするにはZIPにアップロードしてから、サーバーよりまた解凍してURLを利用するのをお勧めします。
